# Oukami

Oukami is flutter manga app with API from Comic Bahasa Indonesia.  

## Features
- [x] Manga List 
- [x] Manhua List 
- [x] Manhwa List
- [x] Last chapter update
- [x] Download Manga
- [x] Favourite Manga
- [x] Search Manga
- [x] History
